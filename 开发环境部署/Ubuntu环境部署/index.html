Ubuntu下开发环境部署

如果要在Ubuntu上进行相应项目开发，需要先进行相应的dotnet环境部署，然后使用跨平台IDE Rider进行开发，由于平台限制，dotnet需要选取.net 6及以上版本。

1、dotnet环境安装

Ubuntu环境下，安装dotnet环境比较复杂，需要根据Ubuntu版本和需要安装的dotnet版本进行区分

Ubuntu 16.04上安装dotnet

在Ubuntu 16.04中，目前仅支持.NET 6，但是不支持Ubuntu源安装，只能使用Microsoft源安装：

  1. 添加 Microsoft 包存储库
wget https://packages.microsoft.com/config/ubuntu/16.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.debbas
  1. 安装SDK
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0

Ubuntu 18.04上安装dotnet

在Ubuntu 18.04中，目前支持.NET 6和.NET 7，但是不支持Ubuntu源安装，只能使用Microsoft源安装：

  1. 添加 Microsoft 包存储库
wget https://packages.microsoft.com/config/ubuntu/18.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
  1. 安装SDK
## .net 6安装
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0

## 或者安装.net 7
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-7.0

Ubuntu 20.04上安装dotnet

在Ubuntu 20.04中，目前支持.NET 6和.NET 7，但是不支持Ubuntu源安装，只能使用Microsoft源安装：

  1. 添加 Microsoft 包存储库
wget https://packages.microsoft.com/config/ubuntu/20.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
  1. 安装SDK
## .net 6安装
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0

## 或者安装.net 7
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-7.0

Ubuntu 22.04上安装dotnet

在Ubuntu 20.04中，目前支持.NET 6和.NET 7，其中.net 6已经在Ubuntu源中受支持，但是.net 7不支持，所以安装方式需要分情况：

  1. 只需要. net 6，不需要.net 7

在只需要.net 6的时候，使用Ubuntu源既可，直接打开终端，运行以下命令：

sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0
  1. 需要.net 7

在需要.net 7的时候， 就需要添加Microsoft源进行安装，如果同时需要安装.net 6的情况下，就需要先屏蔽Ubuntu源中的.net支持，都使用Microsoft安装，如果同时使用两个源进行安装，会出现问题。

屏蔽Ubuntu中dotnet源支持：

① 先删除已经安装的.net版本

sudo apt remove 'dotnet*' 'aspnet*' 'netstandard*'

② 创建/etc/apt/preferences文件

touch /etc/apt/preferences

③ 在编辑器中打开 /etc/apt/preferences 并添加以下设置，以防止以 dotnet、 aspnetcore或 netstandard 开头的包源自分发的存储库

Package: dotnet* aspnet* netstandard*
Pin: origin "<your-package-source>"
Pin-Priority: -10

其中<your-package-source>需要替换为自己的源，如我使用的是中科大的源，则我填入的就是mirrors.ustc.edu.cn,我的文件如下：

Package: dotnet* aspnet* netstandard*
Pin: origin "mirrors.ustc.edu.cn"
Pin-Priority: -1

④ 添加 Microsoft 包存储库

# Get Ubuntu version
declare repo_version=$(if command -v lsb_release &> /dev/null; then lsb_release -r -s; else grep -oP '(?<=^VERSION_ID=).+' /etc/os-release | tr -d '"'; fi)

# Download Microsoft signing key and repository
wget https://packages.microsoft.com/config/ubuntu/$repo_version/packages-microsoft-prod.deb -O packages-microsoft-prod.deb

# Install Microsoft signing key and repository
sudo dpkg -i packages-microsoft-prod.deb

# Clean up
rm packages-microsoft-prod.deb

# Update packages
sudo apt update

⑤ 安装SDK

## .net 6安装
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0

## 或者安装.net 7
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-7.0

Ubuntu 22.10上安装dotnet

在Ubuntu 22.10中，目前支持.NET 6和.NET 7，且都支持Ubuntu源的安装，所以直接使用命令安装既可：

## .net 6安装
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-6.0

## 或者安装.net 7
sudo apt-get update && \
  sudo apt-get install -y dotnet-sdk-7.0

2、Avalonia相关项目模板安装

当dotnet SDK安装完成后，就可以进行Avalonia项目的开发，但是Avalonia有其特殊的目录结构，所以他提供了一些项目模板，供我们方便快捷的进行项目创建：

如果已经安装了.net 7 SDK，则运行以下命令进行项目模板安装：

dotnet new install Avalonia.Templates

如果只安装了.net 6未安装.net 7,则需要运行以下命令来进行安装：

dotnet new -i Avalonia.Templates

3、开发利器，Rider安装

Rider是JetBrains推出的一款跨平台.net IDE，它对Avalonia具有天然的支持，我们强烈建议您使用Rider进行开发，在Linux上，目前也仅有这一款软件可以进行Avalonia开发（在Windows上，Visual Studio也支持），不过它是收费软件，提供30天的试用期，您可以试用后进行购买（Rider也会有预览版推出，不需要收费，并且可以体验Rider一些尚在开发的新功能）。

在Ubuntu上进行Rider安装非常简单，因为Rider已经存在于Ubuntu的snap商店中，打开Ubuntu的Ubuntu Software软件，搜索Rider进行安装即可。

Ubuntu在16.04 LTS及以后版本具有Snap安装软件功能

4、可视化设计器安装

当上述步骤都完成以后，距离开发我们就只差一步，Avalonia插件安装，这个插件为Rider提供了可视化设计器的功能，有了这个插件，您就可以在Rider里面享受在VS中开发WPF/Winform一样的开发体验。

首先打开您的Rider软件，在项目选择界面，找到左下角的Configure按钮，打开下拉框，打开Plugins界面 在Plugins界面中，搜索Avalonia,然后找到AvaloniaRider插件，点击进行安装，然后按照要求重启。

进行完上面这几步后，恭喜您已经完成整个开发环境的搭建，接下来就可以进行项目开发了。
